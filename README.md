# pencatatan-keuangan-app
mencatat, mengelola, dan memantau pemasukan serta pengeluaran pribadi agar pengguna dapat mengontrol keuangan


## fitur utama
- Tambah dan catat pemasukan & pengeluaran 📝
- Kategori transaksi (makanan, transport, dll) 🗂️
- Riwayat transaksi 🕐
- Ringkasan keuangan (harian/bulanan) 📊
- Grafik laporan keuangan 📈
- Edit & hapus transaksi


## tech
- framework : flutter
- database  : sqflite
